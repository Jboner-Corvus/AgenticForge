# Tmux Cheatsheet

## Sessions
- `tmux new-session -s <name>` : Créer une nouvelle session avec un nom
- `tmux ls` : Lister les sessions
- `tmux attach -t <name>` : Attacher à une session
- `tmux detach` (ou `Ctrl+b d`) : Détacher de la session courante
- `tmux kill-session -t <name>` : Tuer une session

## Windows (onglets)
- `Ctrl+b c` : Créer une nouvelle fenêtre
- `Ctrl+b w` : Lister les fenêtres
- `Ctrl+b n` : Passer à la fenêtre suivante
- `Ctrl+b p` : Passer à la fenêtre précédente
- `Ctrl+b <numéro>` : Aller à une fenêtre spécifique
- `Ctrl+b ,` : Renommer la fenêtre courante
- `Ctrl+b &` : Fermer la fenêtre courante

## Panes (volets)
- `Ctrl+b %` : Diviser verticalement
- `Ctrl+b "` : Diviser horizontalement
- `Ctrl+b <flèches>` : Naviguer entre les panes
- `Ctrl+b z` : Maximiser/restaurer un pane
- `Ctrl+b x` : Fermer le pane courant
- `Ctrl+b !` : Convertir le pane en fenêtre

## Commandes utiles
- `Ctrl+b ?` : Afficher l'aide
- `Ctrl+b :` : Entrer une commande tmux
- `Ctrl+b [` : Entrer en mode copie (navigation avec les flèches, q pour quitter)
